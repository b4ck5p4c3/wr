wr public api

These are the read-only endpoints a member site uses to navigate the ring. The panel, the admin, and the auth endpoints are internal and are not part of this contract.

GET /sites

The reachable member sites, in ring order.

[
  { "slug": "a", "name": "Site A", "url": "https://a.example", "favicon": "" }
]

GET /{slug}

A redirect to the member site for the slug.

302 Location: https://a.example/

GET /{slug}/{step}

A redirect to a neighbor in the ring. The step is one of next, prev, or random. 

302 Location: https://b.example/

GET /{slug}/data

The previous, current, and next sites for the slug, without a redirect. The stepped form /{slug}/{step}/data returns the same shape around the stepped position. 

{
  "previous": { "slug": "b", "name": "Site B", "url": "https://b.example", "favicon": "" },
  "current":  { "slug": "a", "name": "Site A", "url": "https://a.example", "favicon": "" },
  "next":     { "slug": "b", "name": "Site B", "url": "https://b.example", "favicon": "" }
}

back to the ring
