wr webring API v1.0.0

The public API. The raw spec is openapi.yaml. Return to the ring.

navigation

GET/sitesList the active sites.
responses
  • 200 array of WebringWebsite
GET/{slug}Redirect to a site.
parameters
  • slug
responses
  • 302 A redirect to the site url.
  • 404 The slug is unknown or the ring is empty.
GET/{slug}/dataReturn the navigation around a site.
parameters
  • slug
responses
  • 200 WebringWebsiteNavigationData
  • 404 The slug is unknown or the ring is empty.
GET/{slug}/{step}Step the ring and redirect.
parameters
  • slug
  • step
responses
  • 302 A redirect to the stepped site url.
  • 404 The slug is unknown or the ring is empty.
GET/{slug}/{step}/dataStep the ring and return the navigation.
parameters
  • slug
  • step
responses
  • 200 WebringWebsiteNavigationData
  • 404 The slug is unknown or the ring is empty.

auth

GET/auth/githubStart the GitHub sign-in.
responses
  • 302 A redirect to the GitHub authorize page.
GET/auth/github/callbackFinish the GitHub sign-in.
parameters
  • code
  • state
responses
  • 302 >-
  • 400 A required field is missing or invalid.
  • 401 The caller is not signed in.
GET/auth/telegram/callbackFinish the Telegram sign-in.
parameters
  • hash
  • id
  • auth_date
  • first_name
responses
  • 302 A redirect with the wr_session cookie set.
  • 400 A required field is missing or invalid.
  • 401 The caller is not signed in.
POST/auth/logoutEnd the session.session
responses
  • 200 A plain message.
  • 405 The endpoint requires POST.
GET/auth/devBypass the login in dev mode.
parameters
  • role
responses
  • 302 A redirect with the wr_session cookie set.
  • 404 The slug is unknown or the ring is empty.
